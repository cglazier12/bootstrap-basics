"col-xl-3 col-2"

Col 3: I have a width of three columns on X-large screens and width of two on smaller screens

"col-xl-3 col-2"

Col 4: I have a width of two columns on all screens

"col-2"

Notice how the widths still change as you make the width of the screen smaller
This is because bootstrap is reactive. As the width of the container changes, so must the width of the row and therefore the width of the columns
